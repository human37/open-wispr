Multiple models

Choose from tiny (75 MB, fastest) to medium (1.5 GB, most accurate). Switch with a single command.

Multilingual

Supports 99 languages via Whisper's multilingual models. Set your language code and go.

Change hotkey

open-wispr set-hotkey globe          # default
open-wispr set-hotkey rightoption   # Right Option
open-wispr set-hotkey f5            # F5
open-wispr set-hotkey ctrl+space    # Ctrl + Space
brew services restart open-wispr

Switch model

open-wispr set-model tiny.en       # 75 MB
open-wispr set-model base.en       # 142 MB (default)
open-wispr set-model small.en      # 466 MB
open-wispr set-model medium.en     # 1.5 GB
brew services restart open-wispr
